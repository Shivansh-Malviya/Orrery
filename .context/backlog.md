# backlog.md

Parking lot for deferred work, reminders, and off-topic ideas.
Promote items to `todo.md` when they become active.

## Format

- [ ] YYYY-MM-DD - [Description] (#tag)
- [x] YYYY-MM-DD - [Completed item] (#tag)

## Backlog

- [ ] YYYY-MM-DD - Item description here. Why deferred: [reason]. (#icebox)
