# inbox (root-level)

Top-level drop zone for the BROS shell. Distinct from workspace/inbox/ which is user content.

## Structure
- .archive/    - Archived inbox content
- attachments/ - Image/screenshot attachments
- code/        - Drop-zone for code (Python, notebooks, etc.)
- docs/        - Drop-zone for documents (PDFs, contracts, etc.)
- exports/     - Exports (ChatGPT, Notion, etc.)
- misc/        - Miscellaneous drop-zone
