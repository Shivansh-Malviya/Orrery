# inbox (root-level)

**Root-level triage inbox.** Distinct from `workspace/inbox/` which is user content.

Top-level drop zone for the BROS shell.

## Structure
- `.archive/` — Archived inbox content
- `attachments/` — Image/screenshot attachments
- `code/` — Drop-zone for code (Python, notebooks, etc.)
- `docs/` — Drop-zone for documents (PDFs, contracts, etc.)
- `exports/` — Exports (ChatGPT, Notion, etc.)
- `misc/` — Miscellaneous drop-zone

## Policy
Inbox content is protected by `policy/OUTPUT_PATH_POLICY.md` — read-only unless explicitly authorized.
