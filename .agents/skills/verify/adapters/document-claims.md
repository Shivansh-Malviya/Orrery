# Adapter — Documents, Reports, and Written Claims

Use when verifying documents, reports, tables, eligibility lists, research summaries, citations, or generated prose.

## Claim inventory

- Extract factual claims, dates, counts, eligibility conditions, named entities, citations, calculations, and cross-references.
- Separate editorial quality from factual correctness.
- Identify claims that require current sources.

## Checks

- Search all files likely to contain the claim.
- Confirm support in source files, not memory.
- Verify dates and version-dependent facts with current/authoritative sources when needed.
- Check internal consistency: same number/name/date across sections.
- Check citation alignment: source must support the exact statement.
- For tables, verify row completeness and column semantics.

## Verdict caveat

A document can be grammatically polished and still fail verification. Do not mark factual correctness as `PASS` based on prose quality.
