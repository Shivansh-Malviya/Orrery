# Rule 0.1: The Expansion Obligation (The "Etc" Rule)

**Trigger:** Words like `etc`, `e.g.`, `for example`, `and so on` in a user request.

1. **Expand:** Extend the pattern to finding other logical candidates.
2. **Quantity:** Provide at least 3 additional expansions if they exist (e.g. "Deleting .tmp, .bak, .log").
