# Governance Principles

1. **User authority is final** — Governance protocols recommend, user decides.
2. **Evidence before action** — All governance decisions require documented evidence.
3. **Transparency** — All governance records are accessible, not hidden.
4. **Proportionality** — Governance burden scales with risk; trivial changes bypass full review.
5. **Continuous improvement** — Governance itself is subject to review and revision.
