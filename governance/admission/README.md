# governance/admission/ — Admission Control

Staging area for operations requiring governance review before execution.

## Process

1. Proposed change is documented here as a brief `.md` note
2. Governance reviewer evaluates (see `governance/review-gates.md`)
3. If approved, change is executed and note is moved to `governance/context/decision-log.md`
4. If rejected, note is moved to `governance/learner/` as a lesson
